\Abstract{
В работе предложен и реализован комплексный подход к оценке
 качества русскоязычных Retrieval-Augmented Generation систем.
  На базе англоязычного бенчмарка RAGBench собран и опубликован его 
  русскоязычный аналог RAGBench-RU (7~поддатасетов, переведённых Qwen2.5-72B). 
  Разработан пайплайн обучения токен-уровневых моделей-оценщиков TRACe с тремя
   классификационными головами (relevance, utilization, adherence) и автоматическим
    подбором порогов. Проведена сетка экспериментов: четыре backbone (DeBERTa-v3-large,
     BGE-M3, RuModernBERT, ruRoPEBert-e5-base-2k), семь длин контекста 
     (128--8192~токенов), две схемы весов лосса 
     (1:1:1 и~3:3:1), два языка (RU/EN) и две архитектуры головы (Simple, Complex) -
      всего около восмидесяти обученных конфигураций. Лучшая конфигурация на русском
       (ModernBERT-base, длина~2048, Simple~3:3:1) достигает
        среднего Test~F1 = 0.858 и уверенно превосходит LLM-оценщик 
        на базе Qwen2.5-72B по token-F1 для relevance и~utilization. 
        Отдельно описан интерактивный сервис оценки на Streamlit и
         проведён анализ переноса подхода на русский язык.
}

\AbstractEng{
This thesis presents a comprehensive framework for evaluating 
Russian-language Retrieval-Augmented Generation systems. Building on
 the English RAGBench benchmark, we translate and release RAGBench-RU 
 (7~sub-datasets, Qwen2.5-72B machine translation). We design a training pipeline
  for token-level TRACe evaluators with three classification heads (relevance, 
  utilization, adherence) and automatic threshold tuning. A grid of experiments 
  spans four backbones (DeBERTa-v3-large, BGE-M3, RuModernBERT, ruRoPEBert-e5-base-2k),
   seven context lengths (128--8192~tokens), two loss-weight schemes (1:1:1 and~3:3:1),
    two languages (RU/EN) and two head architectures (Simple, Complex) -
     around eighty trained configurations in total. The best Russian 
     configuration (ModernBERT-base, length~2048, Simple~3:3:1) reaches mean 
     Test~F1 = 0.858 and outperforms a Qwen2.5-72B LLM-based evaluator on 
     token-F1 for relevance and utilization. An interactive Streamlit service 
     and an analysis of cross-lingual transfer complete the study.
}

    name=Абляционное исследование,
    text=абляционное исследование,

}
\newglossaryentry{промпт}{
    name=Промпт,
    text=промпт,
    description={текстовая инструкция-запрос к большой языковой модели, задающая формат ответа, роль и ограничения; основной инструмент получения нужного поведения от LLM без дообучения}
}
\newglossaryentry{логит}{
    name=Логит,
